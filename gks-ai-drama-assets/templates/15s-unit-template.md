# 15秒叙事单元模板

> 用途：把剧本拆成适合 AI 视频生成的小段落。  
> 原则：每个单元只做一件事。别让模型同时演戏、转场、变装、打斗、煽情，它不是你剧组里的八爪鱼。

---

## 单元基础信息

```txt
单元编号：
所属项目：
预计时长：15秒 / 10秒 / 5秒 / 其他
所属场景：
出现人物：
镜头等级：A自由生成 / B半控制 / C强控制
```

---

## 叙事内容

```txt
这一段发生了什么：
核心动作：
情绪起点：
情绪终点：
进入点：
退出点：
```

示例：

```txt
这一段发生了什么：
男主发现火车即将离开，开始追赶火车。

核心动作：
从站台中段向前奔跑，最后停下。

情绪起点：
压抑、迟疑。

情绪终点：
急切、崩溃但克制。

进入点：
火车开始缓慢移动。

退出点：
男主停在站台尽头，低头喘息。
```


示例：

```txt

```
