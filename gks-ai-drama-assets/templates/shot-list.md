# 镜头表模板

> 用途：管理 AI 剧 / AI 短片 / AI MV 的镜头。  
> 建议：不要把所有镜头都写成 C 级强控制，不然项目会变成炼丹地狱。

---

## 镜头等级说明

### A 类：自由生成

适合：

- 空镜
- 氛围
- 环境
- 抽象画面
- 情绪过渡
- 光影变化

控制方式：

```txt
文本提示词直接生成
```

---

### B 类：半控制

适合：

- 普通人物动作
- 普通剧情推进
- 风格统一镜头
- 不要求精确动作的镜头

控制方式：

```txt
文本提示词 + 人物参考图 / 场景参考图
```

---

### C 类：强控制

适合：

- 人物特写
- 核心表演
- 关键道具
- 产品展示
- 复杂动作
- 结尾画面
- 海报级画面

控制方式：

```txt
人物资产 + 场景母版 + 首帧控制 + 明确镜头提示
```
