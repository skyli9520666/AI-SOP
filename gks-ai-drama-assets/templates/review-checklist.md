# AI视频检查清单

> 用途：发布前、交付前、归档前，用这份表快速过一遍。  
> 原则：不要因为“生成出来很不容易”就舍不得删。烂镜头不会因为你心疼而变好。

---

## 1. 画面检查

### 人物

- [ ] 人物没有明显变脸
- [ ] 年龄没有明显变化
- [ ] 发型基本一致
- [ ] 服装基本一致
- [ ] 身材比例没有明显漂移
- [ ] 表情符合剧情
- [ ] 没有多出无关人物
- [ ] 没有人物肢体严重错误
- [ ] 手部没有严重畸形，尤其是特写镜头

### 场景

- [ ] 场景空间基本连续
- [ ] 光源方向没有明显跳变
- [ ] 道具没有突然消失
- [ ] 关键物件位置合理
- [ ] 没有错误文字或乱码
- [ ] 没有现代元素误入旧时代场景
- [ ] 没有不符合风格的背景物

### 镜头

- [ ] 景别变化合理
- [ ] 运镜不晕
- [ ] 镜头没有无意义乱动
- [ ] 关键画面停留时间足够
- [ ] 没有明显穿帮
- [ ] 没有严重闪烁
- [ ] 没有明显压缩感破坏观感

---

## 2. 剪辑检查

- [ ] 开头 3 秒能抓住注意力
- [ ] 镜头顺序清楚
- [ ] 情绪有起伏
- [ ] 节奏不拖沓
- [ ] 转场不突兀
- [ ] 空镜使用合理
- [ ] 没有连续展示模型缺陷
- [ ] 每个镜头都服务故事或情绪
- [ ] 没有只因为“好看”但无意义的镜头
- [ ] 结尾画面有记忆点

---

## 3. 声音检查

- [ ] 音乐风格匹配画面
- [ ] 音乐音量不过大
- [ ] 对白或旁白清晰
- [ ] 环境声匹配场景
- [ ] 关键动作有音效支撑
- [ ] 转场音效不过度
- [ ] 没有突然爆音
- [ ] 没有明显底噪问题
- [ ] 手机外放也能听清核心内容
- [ ] 耳机听没有刺耳频段

---

## 4. 字幕检查

- [ ] 字幕没有错字
- [ ] 字幕时间轴准确
- [ ] 字幕位置不挡关键画面
- [ ] 字幕颜色清晰可读
- [ ] 字体风格符合整体视觉
- [ ] 中英文标点统一
- [ ] 人名、平台名、工具名拼写正确
- [ ] 字幕没有过长导致观众读不完

---

## 5. 画质提升检查

- [ ] 分辨率符合发布平台
- [ ] 插帧后动作没有变怪
- [ ] 超分后没有塑料感
- [ ] 人脸修复没有修成另一个人
- [ ] 降噪没有抹掉细节
- [ ] 锐化不过度
- [ ] 黑场没有严重色块
- [ ] 输出后没有二次压缩过重

---

## 6. 平台适配检查

### 横版 16:9

- [ ] 适合 B站 / YouTube / 路演 / 围炉会展示
- [ ] 字幕在投影上可读
- [ ] 关键画面没有太靠边
- [ ] 音量适合现场播放

### 竖版 9:16

- [ ] 适合抖音 / 小红书 / 视频号
- [ ] 主体在中间安全区
- [ ] 字幕没有贴底太低
- [ ] 开头信息足够快

### 方版 1:1

- [ ] 适合部分社交平台信息流
- [ ] 主体构图没有被裁坏
- [ ] 字幕没有被平台 UI 遮挡

---

## 7. 版权与合规检查

- [ ] 音乐来源明确
- [ ] 字体来源明确
- [ ] 图片参考来源明确
- [ ] 没有未经授权使用真人肖像
- [ ] 没有明显侵权角色或品牌
- [ ] 没有误导性新闻画面
- [ ] 没有需要额外授权的素材
- [ ] 是否需要标注 AI 生成
- [ ] 是否需要标注工具来源

---

## 8. 文件与归档检查

- [ ] 最终文件命名清晰
- [ ] 源工程已保存
- [ ] 生成提示词已保存
- [ ] 人物资产已保存
- [ ] 场景资产已保存
- [ ] 最终成片已备份
- [ ] 使用平台和参数已记录
- [ ] 失败案例已记录
- [ ] 可复用工作流已记录

## 10. 一句话判断

发布前最后问一句：

```txt
如果观众不知道你生成这个镜头有多难，他还会觉得它好吗？
```

如果答案是否定的，删掉或重做。
