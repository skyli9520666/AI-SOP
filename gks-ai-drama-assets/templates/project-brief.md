## 3. 一句话概念

用一句话说清楚作品。

```txt
这是一个关于【人物】在【场景】中，因为【冲突】，最终走向【情绪或结果】的故事。
```

示例：

```txt
这是一个关于年轻音乐人在雨夜站台追赶火车，因为没能把旧磁带交给曾经的爱人，最终接受遗憾的故事。
```

## 5. 视觉方向

```txt

时代感：
材质关键词：
参考作品或参考图：
禁止出现的视觉元素：
```

示例：

```txt
整体气质：克制、孤独、潮湿、独立电影感
色彩关键词：黑灰、深蓝、低饱和暖黄
光线关键词：雨夜反光、远处车灯、昏暗站台灯
摄影风格：浅景深、轻微手持、胶片颗粒
禁止元素：明亮商业广告感、赛博朋克霓虹、现代高铁站
```
