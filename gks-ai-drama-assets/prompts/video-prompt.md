# 视频生成提示词模板

> 用途：把剧本 / 分镜 / 15秒叙事单元转成视频模型更容易执行的提示词。  
> 原则：提示词不是形容词堆叠，而是导演说明。

---

## 1. 通用视频提示词结构

```txt
主体：
场景：
动作：
情绪：
镜头：
光线：
风格：
时长：
禁止事项：
```

---

## 2. 通用模板

```txt
主体：
【角色身份、年龄、外貌、服装、气质、关键道具】

场景：
【时间、地点、天气、空间结构、环境细节】

动作：
【角色从哪里开始，做什么，动作如何变化，在哪里结束】

情绪：
【情绪起点 → 情绪终点，表演程度，是否克制】

镜头：
【景别、机位、运镜、构图、镜头节奏】

光线：
【主光源、色温、明暗关系、反光、阴影】

风格：
【电影类型、质感、色彩、颗粒、摄影风格】

时长：
【5秒 / 10秒 / 15秒】

禁止事项：
【不要出现的元素、不要的风格、不要的动作、不要的错误】
```

## 4. 示例：雨夜站台

```txt
请生成一个约 15 秒的视频片段。

主体：
一位年轻男音乐人，黑色中长发，清瘦，穿深色旧外套，手里攥着一盘旧磁带。

场景：
深夜雨中的老式火车站台，远处火车即将离开，站台灯光昏暗，地面有积水反光。

动作：
他从站台中段开始追赶火车，脚步越来越急，最后停在站台尽头，低头喘息。

情绪：
从压抑和迟疑，逐渐变成急切，最后是克制的崩溃。不要夸张哭喊。

镜头：
从远景缓慢推进到中近景，轻微手持感，最后停在他的侧脸和手中的磁带上。

光线：
低饱和夜景，冷色雨夜，远处火车灯带来微弱暖光，雨水反射站台灯。

风格：
独立爱情电影，胶片颗粒，克制构图，真实光线，低饱和色彩。

节奏：
前半段缓慢压抑，中段加速，最后突然停顿。

禁止事项：
不要现代高铁站，不要赛博朋克霓虹，不要商业广告质感，不要人物突然变脸，不要夸张动作，不要错误文字。
```

## 5. 平台适配提示词

### 5.1 让 LLM 改写成不同平台版本

```txt
请将下面这个镜头提示词分别改写成适合不同平台的视频生成提示词。

平台包括：
1. 即梦 / Seedance
2. 可灵
3. Google Flow
4. 通用图生视频模型

要求：
- 保持同一画面意图。
- 根据平台特点调整语言结构。
- 不要加入新的剧情。
- 保留禁止事项。
- 每个平台输出一个版本。

原始提示词：
【粘贴提示词】
```

## 6. 文生视频模板

适合没有参考图，快速测试方向。

```txt
请生成一个【时长】的视频。

画面内容：
【主体 + 场景 + 动作】

镜头语言：
【景别 + 运镜 + 构图】

视觉风格：
【色彩 + 光线 + 材质 + 电影类型】

情绪：
【情绪关键词 + 表演强度】

禁止：
【不要出现的元素】
```

## 7. 图生视频模板

适合已经有人物图、场景图、首帧图。

```txt
请基于参考图生成一个【时长】的视频。

保持参考图中的：
- 人物五官
- 服装
- 场景结构
- 色彩氛围
- 构图关系

动作：
【角色动作】

镜头：
【运镜和节奏】

情绪：
【情绪变化】

禁止：
不要改变人物身份，不要改变服装，不要改变场景结构，不要新增无关人物，不要夸张变形。
```

## 8. 首帧控制模板

```txt
接下来的动作：
【动作描述】

【镜头怎么动】

情绪变化：
【情绪变化】

结束画面：
【希望最后停在哪里】

禁止事项：
【不要出现什么】
```

## 9. 失败结果复盘提示词

```txt
我用下面提示词生成的视频失败了。

原提示词：
【粘贴提示词】

失败现象：
【描述失败结果，例如人物变脸、动作怪、场景不连续、画面太广告感】

请帮我判断：
1. 失败可能来自提示词哪里？
2. 哪些内容过度复杂？
3. 哪些内容需要参考图或首帧控制？
4. 如何改写提示词？
5. 是否建议拆成多个镜头？
6. 给我一个更稳的新版提示词。
```

## 10. 统一视觉后缀模板

### 独立电影感

```txt
低饱和度独立电影质感，克制构图，真实光线，浅景深，胶片颗粒，冷暖对比，人物情绪含蓄，镜头语言克制，不要广告感，不要过度美颜。
```

### 科技控制台感

```txt
黑底红橙荧光视觉，细网格，工程制图感，CRT扫描线，低亮度高对比，复古航天控制台界面，克制硬朗，不要廉价赛博朋克，不要彩虹渐变。
```

### 潮湿雨夜感

```txt
深色雨夜，低饱和蓝灰色调，潮湿空气，地面积水反光，远处暖色灯光，胶片颗粒，轻微手持镜头，孤独克制的情绪。
```

### 梦境回忆感

```txt
柔和低对比光线，轻微失焦，慢速镜头，浅景深，记忆碎片感，色彩温柔但不甜腻，画面有轻微漂浮感，不要过度梦幻特效。
```
